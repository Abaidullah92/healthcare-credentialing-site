# 📘 Quad Solutions — Medical Credentialing Website

## 📌 Project Overview

**Quad Solutions** is a modern, responsive website designed for a medical credentialing service provider. The platform showcases services, builds trust with healthcare clients, and allows users to book a free consultation through an interactive multi-step form.

This project was developed as part of an internship to demonstrate front-end development skills, UI/UX design, and real-world business website implementation.

---

## 🚀 Key Features

### 🌐 Main Website
- Professional landing page with hero section
- Service showcase (Credentialing, Billing, Compliance)
- Step-by-step process timeline
- Testimonials and reviews section
- FAQ accordion
- Contact form with validation
- Responsive navigation (mobile-friendly)
- Trust indicators (HIPAA, success rate, etc.)

### 📅 Booking System
- Multi-step form (4 steps):
  1. User Information  
  2. Service Selection  
  3. Schedule Selection  
  4. Confirmation  
- Progress tracking UI
- Date & time slot selection
- Form validation
- Review before submission
- Success confirmation screen

### 🎨 UI/UX Design
- Modern healthcare-focused design
- Fully responsive (desktop, tablet, mobile)
- Smooth animations and transitions
- Clean typography (Google Fonts: Fraunces & Inter)
