# 📘 Quad Solutions — Medical Credentialing Website

## 📌 Project Overview

**Quad Solutions** is a modern, responsive website designed for a medical credentialing service provider. The platform showcases services, builds trust with healthcare clients, and allows users to book a free consultation through an interactive multi-step form.

This project was developed as part of an internship to demonstrate front-end development skills, UI/UX design, and real-world business website implementation. The website is fully functional, HIPAA-compliant-ready, and optimized for desktop and mobile devices.

---

## 🚀 Key Features

### 🌐 Main Website (Home Page)

- **Professional landing page** with hero section highlighting key value propositions
- **Service showcase** featuring Credentialing, Billing, and Compliance services
- **Process timeline** with step-by-step explanation of the credentialing workflow
- **Testimonials & reviews** section to build trust with prospective clients
- **Interactive FAQ accordion** for common questions
- **Contact form** with client-side validation
- **Responsive mobile navigation** with hamburger menu
- **Trust indicators** (HIPAA compliant, 95% approval rate, 24/7 portal)

### 📅 Consultation Booking System

- **Multi-step form** (4 interactive steps):
  1. User Information (name, email, phone)
  2. Service Selection (choose services needed)
  3. Schedule Selection (date & time slots)
  4. Confirmation (review and submit)
- **Progress tracking** showing current step and completion status
- **Interactive date/time picker** for flexible scheduling
- **Form validation** for all required fields
- **Review system** before final submission
- **Success confirmation screen** with next steps

### 🎨 UI/UX Design

- **Modern, healthcare-focused design** with professional color scheme
- **Fully responsive layout** optimized for desktop, tablet, and mobile
- **Smooth transitions and animations** for enhanced user experience
- **Professional typography** using Google Fonts (Fraunces & Inter)
- **Clean information architecture** with clear sections and CTAs

## 🎯 Getting Started

### Prerequisites

- No build tools required!
- Any modern web browser (Chrome, Firefox, Safari, Edge)

### Installation & Running

1. **Extract the project folder**

   ```bash
   Extract all files to your desired location
   ```

2. **Open in browser**
   - Double-click `index.html` to open the homepage
   - Click "Book Free Call" to navigate to the booking page
   - All files are self-contained (no server setup needed)

3. **File structure requirements**
   - Keep all files (`index.html`, `book-consultation.html`, `styles.css`, `script.js`) in the same directory
   - Ensure the `images/` folder is in the same directory with all image assets

### Local Server (Optional)

For best results with certain features, run a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

Then visit `http://localhost:8000`
